# Agent Onboarding: Day 1
### How to run the Austin Deal Desk in 5 minutes.

Welcome to the team. We don't use a complex CRM. We use **The Desk**. 
Follow these four steps to handle your first client request.

---

## 1. The Golden Rule: No Envelope, No Work
If a client texts you a request, it doesn't exist until it’s in a **Manila Envelope**.
- Go to `MANILA_ENVELOPE.md`.
- Copy the template.
- Create a new file named `AUSTIN-YYYYMMDD-CLIENTNAME.md`.
- Fill in what you know. Mark what you don't know as `Unknown`.

## 2. Start at the Orchestrator (Station 00)
Move your new envelope into the `00_orchestrator` folder. 
The AI will read the envelope and tell you which station needs to handle it next. 

## 3. Respect the Slips
The system uses colored slips to flag risk. You must check these before every client call:
- **RED:** Stop everything. Diana needs to see this now.
- **YELLOW:** Information is missing. Do not guess.
- **BLUE:** Compliance or verification required. Check `AUSTIN_RESOURCES.md` for where to verify.
- **GREEN:** The file is clean. Move to the next station.

## 4. The "Blue Slip" Hard Stop
If you are trying to schedule a showing and the envelope has a **BLUE Slip** on Buyer-Representation:
**DO NOT CONFIRM THE SHOWING.** The system is designed to block you here to protect the team. Send the envelope to Diana for verification. Once she clears the flag, the "Green Slip" will appear.
