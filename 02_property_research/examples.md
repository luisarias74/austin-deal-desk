# 02 Property Research Examples

## Example 1: 1938 Tarrytown Bungalow

### Situation

Buyer is interested in a 1938 bungalow in Tarrytown.

### Action

Flag:

- older home risk
- pier-and-beam/foundation review
- possible historic/zoning verification
- school/neighborhood claim verification, if client-facing use is planned

### Desk Trail

```md
[02-PR] 2026-05-12 — Reviewed older Tarrytown property. Applied YELLOW slip for older-home/foundation risk and BLUE slip for historic/zoning verification before client-facing guidance.
```
