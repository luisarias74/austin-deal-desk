# Austin Resources

A practical verification index for BLUE slips.

Verify destination links and current form/version numbers before each use. Resources change over time. This index is a starting point, not a final authority.

This file does not provide legal conclusions. Use the language:

- verify
- check current source
- flag for Diana/broker/licensed professional

---

## Buyer Representation, IABS, And Contract Forms

**Destination:** TREC (Texas Real Estate Commission)

**Use when a BLUE slip involves:**

- buyer representation agreement language
- IABS (Information About Brokerage Services) handling
- current TREC residential contract form (e.g., 1-4 Family Residential)
- NAR-settlement written-agreement compliance practice

**How to use:** Check the current TREC source for the most recent forms. Flag for Diana/broker/licensed professional before relying on form interpretation.

## Property Records, Ownership, Square Footage, Year Built

## Floodplain

**Destination:** FEMA Flood Map Service Center and local floodplain sources

**Use when a BLUE slip involves:**

- FEMA flood zone designation
- local floodplain or watershed concern
- creek, lake, or drainage adjacency
- elevation certificate question

**How to use:** Check the current FEMA source and any local floodplain source. Verify against the current parcel. Flag for Diana/broker/licensed professional before client guidance.

## Foundation, Structural, Engineering Concerns

**Destination:** Inspector / engineer / licensed professional review

**Use when a BLUE slip involves:**

- pier-and-beam foundation concern
- slab movement or cracking
- Central Texas soil/foundation concern
- structural integrity question
- moisture intrusion question

**How to use:** The Deal Desk does not make engineering conclusions. Flag for Diana/broker/licensed professional. Recommend a qualified inspector or structural engineer for verification.

## How To Use This File With BLUE Slips

When applying a BLUE slip, name the destination from this file when possible.

Example:

```md
BLUE slip — historic/zoning verification needed.
See AUSTIN_RESOURCES.md → City of Austin zoning and historic preservation resources.
```

This keeps BLUE slips actionable instead of vague.
